# 音乐推荐平台 python773

本项目是基于Python实现的音乐推荐系统，使用MySQL数据库存储数据，并以Django框架进行后端开发，整合了爬虫技术以获取音乐信息。

## 技术栈

- **后端语言**: Python
- **开发框架**: Django
- **数据库**: MySQL
- **数据获取**: 爬虫技术

## 功能模块

- 音乐信息获取
- 用户行为分析
- 音乐推荐算法
- 用户界面展示

## 系统角色

- **用户**: 普通用户，可以浏览、搜索和收藏音乐
- **管理员**: 系统管理者，负责音乐信息的审核和系统维护

## 运行环境

- Python 3.x
- MySQL 5.x 或更高版本
- 操作系统兼容性：支持Linux和macOS，理论上也兼容Windows

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 克隆项目代码
4. 安装依赖库（根据`requirements.txt`）
5. 配置数据库连接
6. 运行Django项目

## 目录结构

```
.
├── app                 # Django应用目录
│   ├── migrations      # 数据库迁移文件
│   ├── models.py       # 数据模型定义
│   ├── views.py        # 视图逻辑
│   └── ...
├── db                 # 数据库相关文件
├── media              # 媒体文件存储
├── static             # 静态文件存储
├── manage.py          # Django项目管理脚本
├── requirements.txt   # 项目依赖
└── ...
```

## 核心亮点

- **智能推荐算法**: 基于用户历史行为和音乐元数据，提供个性化推荐
- **高效数据爬取**: 实时获取互联网上的音乐信息，确保内容丰富性
- **易于扩展**: 基于Django框架，方便后续的功能扩展和二次开发
